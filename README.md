## ğŸ® Controls

### Laptop Keyboard Mapping
This app turns your computer keyboard into a musical instrument. The keys are mapped in two rows to simulate a real piano layout.

| Row | Keys Used | Function |
| :--- | :--- | :--- |
| **Lower Row (White Keys)** | `Z` `X` `C` `V` `B` `N` `M` ... | Plays natural notes (Sa, Re, Ga...) |
| **Upper Row (Black Keys)** | `S` `D` `G` `H` `J` ... | Plays sharp/flat notes (Komal/Tivra) |

**Tip:** Click the **"View: Laptop Keys"** button in the app to see exactly which letter corresponds to which note on the piano keys!
