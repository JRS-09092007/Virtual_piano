## ğŸ® Controls

### Laptop Keyboard Mapping
This app maps your computer keyboard to the piano keys in a two-row layout, allowing you to play chords easily.

| Row | Keys Used | Function |
| :--- | :--- | :--- |
| **Lower Row (White Keys)** | `Z` `X` `C` `V` `B` `N` `M` ... | Plays natural notes (White Keys) |
| **Upper Row (Black Keys)** | `S` `D` `G` `H` `J` ... | Plays sharp/flat notes (Black Keys) |

**Tip:** Click the **"View: Laptop Keys"** button in the app to see exactly which letter corresponds to which note on the piano!
